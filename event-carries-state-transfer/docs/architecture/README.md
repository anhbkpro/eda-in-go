# Architecture Documentation (C4 Model)

This directory contains C4-style architecture diagrams using PlantUML.

## Structure

- **Context**: System boundary and external actors
- **Containers**: Microservices and data flows
- **Components**: Internal structure of key services

## How to Render

- VS Code: Install *PlantUML* extension
- Requires Graphviz
- Open `.puml` files and preview

## Diagram Levels

1. System Context
2. Container Diagram
3. Component Diagram
